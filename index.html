Fix HP Mini Brightness on Ubuntu 14.04

The screen brightness control current laptop, an HP Mini with Ubuntu 14.04 is not working. This is a problem because I sometimes use my laptop late at night. Found a fix by updating the grub.

The solution

  1. Open Terminal
  2. Edit the grub file sudo vim /etc/default/grub
  3. Locate the GRUB_CMDLINE_LINUX_DEFAULT ="quiet splash"
  4. Edit the line to be GRUB_CMPLINE_LINUX_DEFAULT="quiet splash pcie_aspm=force acpi_backlight=vendor"
  5. Update the grub sudo update-grub

Done.

Create and Publishing Post on Octopress

I tried the basic command to create post and publish the post, but it failed. The rake preview is working fine, I can see the update on localhost:4000 but if I access dwiperdana.github.io I can only see an empty blog.

The command I used:

rake new_post["post-name"]
rake generate
rake preview

The actually working command:

rake new_post["post-name"]
rake integrate
rake gen_deploy

After that, my blog post is instantly on the dwiperdana.github.io.
